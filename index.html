About Me

Short story about me

Halo! Nama saya Andi. Saya mulai tertarik pada pemrograman web saat duduk di SMA. Awalnya, saya belajar melalui berbagai forum dan tutorial online, kemudian memperdalam pengetahuan saya melalui video-video di YouTube. Dalam proses ini, saya merasa sangat antusias untuk terus mengembangkan keterampilan dan mengeksplorasi lebih jauh mengenai pemrograman web.

Selain pemrograman, saya juga memiliki minat besar di bidang geografi, khususnya dalam Sistem Informasi Geografis (SIG) dan penginderaan jauh. Saya sangat tertarik untuk memperdalam pemahaman mengenai SIG dan aplikasi pemrogramannya dalam konteks web. Saya yakin bahwa menggabungkan pemrograman dengan SIG akan membuka peluang baru yang sangat menarik, dan saya bersemangat untuk terus belajar serta mengasah kemampuan saya di kedua bidang ini.

Melalui situs ini, saya berharap dapat membagikan pemikiran, hasil proyek, dan pembelajaran saya. Saya percaya bahwa mendokumentasikan perjalanan ini adalah cara terbaik untuk memperkuat pemahaman saya, sekaligus memberikan manfaat bagi orang lain yang memiliki minat serupa. Jangan ragu untuk menghubungi saya, saya akan senang membantu dan berbagi ilmu!
